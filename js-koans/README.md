# Instructions

## Requirements
- Node = v16.19.1
- NPM = 8.19.3
- Yarn = 1.22.19

## Setup

```shell
# cd into a koan directory
$ nvm use lts/gallium
$ yarn install
```

## Creating new koan
- Make new directory with name of koan
- Copy over gitignore, package.json, README.md
- Update README.md
- Run `yarn install`
